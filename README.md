# 二等ドローン学科クイズ／問題集

二等無人航空機操縦士（ドローン国家資格）の **学科試験対策用オリジナル問題集（全200問・三肢択一式）** と、ブラウザで解けるインタラクティブな **学習クイズ** です。

> ⚠️ 本教材は学習用に独自作成したものであり、**実際の試験問題とは異なります**。最新の制度・法令は必ず国土交通省の公式情報でご確認ください。

## 🎯 オンラインで挑戦

GitHub Pages で公開しています。

**👉 https://technodronekurashiki.github.io/drone-license-workbook/**

- 科目（規則／システム／操縦者・運航体制／リスク管理）と出題数を選んで開始
- 1問ずつ解答 → その場で正誤・解説を表示
- 最後にスコアと「不正解の復習」を表示

## 📚 出題範囲と構成

二等無人航空機操縦士 学科試験の科目に沿って 200 問を作成しています。

| 科目 | 問題数 | 割合 |
|---|---:|---:|
| ① 無人航空機に関する規則 | 80 | 40% |
| ② 無人航空機のシステム | 48 | 24% |
| ③ 無人航空機の操縦者及び運航体制 | 36 | 18% |
| ④ 運航上のリスク管理 | 36 | 18% |
| **合計** | **200** | **100%** |

- 形式：三肢択一式（「正しいものを1つ」／「誤っているものを1つ」を約半々）
- 正解番号は 1・2・3 をほぼ均等に配分
- 全200問について論点の重複がないことを確認済み

## 📝 問題集のバリエーション

- **セット完結型**（`_001-050` 〜 `_151-200`、`_全200問_統合版`）
  各問のすぐ下に正解・解説。学習・確認向け。
- **解答分離型**（`_全200問_解答分離型`）
  問題1〜200 を先に並べ、末尾に「解答・解説」をまとめて掲載。模試形式向け。

Markdown 版・Word 版のどちらも内容は同一です。

## 🔐 ログイン機能（Supabase）

会員登録（メール／パスワード／氏名）＋メール認証（マジックリンク）＋パスワード再設定＋管理画面を備えています。
認証基盤は **Supabase**（無料枠あり）を利用し、フロントは GitHub Pages のまま動作します。

- 利用にはログインが必要（未ログインではクイズを表示しません）
- パスワード要件：8文字以上・大文字・数字・記号を各1つ以上
- 管理者は `管理者画面.html` で登録アカウントを一覧／CSV出力できます

**セットアップは [docs/SETUP_AUTH.md](docs/SETUP_AUTH.md) を参照**してください
（Supabaseプロジェクト作成 → `config.js` に接続情報 → `db/schema.sql` 実行 → メール設定）。

